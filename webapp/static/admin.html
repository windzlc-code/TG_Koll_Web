舆情 Cookie 授权

人工维护 Threads、Instagram、X 等平台的浏览器授权 Cookie。后台只显示状态和 Cookie 名称，不回显 Cookie 值。
授权助手流程
网页不能直接安装未上架扩展；先下载并解压助手，在扩展管理页加载该目录，再登录平台并用助手同步 Cookie。
  1. 下载授权助手 zip，解压到本地文件夹。
  2. 打开 Chrome / Edge 扩展管理页，开启开发者模式，选择“加载已解压的扩展程序”。
  3. 点击对应平台的“打开”，完成平台登录。
  4. 在浏览器右上角打开授权助手，点“同步当前站点”。
