授权助手流程
网页不能静默替换浏览器扩展代码；开发环境建议加载项目里的固定助手目录，后续只需在扩展管理页点击刷新。
  1. 打开 Chrome / Edge 扩展管理页，开启开发者模式。
  2. 选择“加载已解压的扩展程序”，加载安装说明中的固定目录。
  3. 点击对应平台的“打开”，完成平台登录。
  4. 在浏览器右上角打开授权助手，点“同步当前页面”。
